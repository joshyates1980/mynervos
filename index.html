MyNervos

is a landing site for the Nervos community. Placeholders represent slots on this site for anyone to obtain. The community can create their custom site to present their involvment with Nervos. This site is open source on GitHub . Fork the master and create a directory in the "u" folder. There is also a staging branch of the master which will test the merge to production. Any static html/javascript site can be hosted on MyNervos.

For assistance on claiming slots and building sites, send a private message on MyNervos Twitter.
